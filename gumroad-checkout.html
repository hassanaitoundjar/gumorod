Gumroad Checkout Implementation

Different ways to implement Gumroad checkout in your website

Method 1: Modal Overlay

Opens a Gumroad product page in a modal on your website

This method uses Gumroad's JS to open a modal overlay with the product details. When the user clicks "Add to cart", they're redirected to Gumroad's checkout page.

<script src="https://gumroad.com/js/gumroad.js"></script>
<a class="gumroad-button" href="https://laradevprof.gumroad.com/l/laraveldev">Buy my product</a>

Method 2: Direct to Checkout

Sends customers directly to the Gumroad checkout page

This method skips the product page and sends users directly to the checkout page. You can also add URL parameters to pre-fill customer information.

<a href="https://laradevprof.gumroad.com/l/laraveldev?wanted=true">Buy Now (Direct to Checkout)</a>

You can also pre-fill customer information using URL parameters:

<a href="https://laradevprof.gumroad.com/l/laraveldev?wanted=true&email=customer@example.com">Buy Now (With Pre-filled Email)</a>

Advanced Options

Additional URL parameters for customizing the checkout experience

You can customize the checkout experience by adding URL parameters to pre-fill customer information:

Parameter Description Example
wanted=true Sends directly to checkout ?wanted=true
email Pre-fills email address &email=customer@example.com
name Pre-fills customer name &name=John%20Doe
referrer Tracks referral source &referrer=yourwebsite
price For pay-what-you-want products &price=20

JavaScript Implementation

Programmatically redirect to checkout

You can also use JavaScript to programmatically redirect customers to the Gumroad checkout page:

<script>
function redirectToCheckout(productId, email = '', name = '') {
    // Base URL for the product
    const baseUrl = `https://laradevprof.gumroad.com/l/${productId}`;
    
    // Add parameters
    const params = new URLSearchParams();
    params.append('wanted', 'true'); // Direct to checkout
    
    if (email) params.append('email', email);
    if (name) params.append('name', name);
    
    // Redirect to checkout
    window.location.href = `${baseUrl}?${params.toString()}`;
}

// Example usage
document.getElementById('checkoutButton').addEventListener('click', function() {
    const email = document.getElementById('emailInput').value;
    const name = document.getElementById('nameInput').value;
    redirectToCheckout('laraveldev', email, name);
});
</script>
